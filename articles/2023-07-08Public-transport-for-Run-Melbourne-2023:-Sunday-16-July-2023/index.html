Public transport for Run Melbourne 2023: Sunday 16 July 2023

Run Melbourne returns in 2023, taking in some of Melbourne’s iconic landmarks such as Melbourne’s Grand Slam Oval, Southbank Promenade, Docklands Harbour and the Botanical Gardens.

Customers are advised that there will be extensive road closures around Melbourne resulting in service changes and will need to plan ahead.

Before you travel

Plan your journey using Journey planner and check Disruptions for service changes that may affect your journey.

 Metropolitan bus

Route 232

Inbound will run via Collins Street to Queen Street and then terminate at Bourke Street.

Outbound Services will then depart Queen Street and run via Bourke Street, William Street to Collins Street from first service to 9:30am.

Outbound services will depart Queen Street and run via Bourke Street to Kings Street and run the reverse route to Crockford Street from first service to 9:30am.
