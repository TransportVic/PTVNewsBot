Public transport during the Easter public holidays

On public holidays, daily myki fare caps apply ($8 full fare and $4 concession/senior), giving you unlimited travel for the day.

For more ticketing information, including myki, regional travel and fare caps, visit: Tickets and myki – Transport Victoria.

Good Friday, 3 April 

Metropolitan train

  • Metro trains will run to a Sunday timetable.
  • Night Network trains will run as normal.

Metropolitan tram

  • Trams run to a Sunday timetable.
  • Night Network trains will run as normal.

Regional train and coach 

  • V/Line trains run to a Sunday timetable. 
    Coaches run to an altered timetable.

Metropolitan and regional bus

  • Metropolitan buses will mostly run to a Sunday timetable.
  • Regional bus services vary by operator.
  • Night network services will run as normal.
  • Check timetables, at: Transport Victoria - Timetables

Easter Saturday, 4 April

Services will run to a normal Saturday timetable. 

Night Network services will run as normal.

Easter Sunday, 5 April

Services will run to a normal Sunday timetable.

Night Network services will not run.

Easter Monday, 6 April

Metropolitan train

  • Metro trains run to a Saturday timetable.
  • Night Network trains will not run.

Metropolitan tram

  • Trams run to a Saturday timetable.
  • Night Network trams will not run.

Regional train and coach

  • V/Line trains will run to a Saturday timetable, except:
    • Swan Hill Line trains which run to a Sunday timetable
    • The 3:49pm Eaglehawk to Southern Cross train departs later at 6:48pm (as it originates from Swan Hill and follows a Sunday timetable)
  • Coaches will run to an altered timetable.

Metropolitan and regional bus

  • Metropolitan buses will mostly run to a Saturday timetable.
  • Regional bus timetables will vary by operator.
  • Night network services will run as normal.
  • Check timetables, at: Transport Victoria - Timetables
