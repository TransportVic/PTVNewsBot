Mobile myki scheduled outage Monday 5 May

Google WalletTM is making some changes to enhance security and improve product features.

To prepare for this change, from 8pm until 8.30pm on Monday 5 May, Mobile myki customers will not be able to top-up or add a Mobile myki to Google WalletTM.

We recommend topping up before this time so you’re ready to travel.

New look Mobile myki 

From Monday 5 May, Android users may see some change to Mobile myki.

New customers will see a new Mobile myki card design. There will also be some minor changes to the settings and how the card looks in the Google Wallet. Customers can still top up myki Money and myki Pass for full fare and general concession.

For more information and help setting up, visit Mobile myki.

If you’re an existing Mobile myki customer, things will stay the same for now. The new look and functionality will be rolled out in the future.
