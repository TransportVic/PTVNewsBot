Public transport for Melbourne Cup Public Holiday

Plan ahead by using Journey planner or by downloading the free PTV app.

Ticketing

  • On public holidays, your daily myki fare is capped at $7.60 ($3.80 for concessions and seniors), giving you unlimited travel all day.
  • While some concession holders receive free travel on weekends, however this does not apply on public holidays.
  • Public holiday fare caps apply to all passengers, including seniors.

For more information, visit Tickets and myki - Transport Victoria.

Metropolitan train

  • Metro trains will run to a Saturday timetable.
  • Night Network services will not run.

We're running extra services to Flemington Racecourse for the 2025 Melbourne Cup Carnival.

Metropolitan tram

  • Trams will run to a Saturday timetable, with the last tram running one hour earlier than usual.
  • Night network services will not run.

Regional train and coach

  • V/Line trains will run to a Saturday timetable.
  • Scheduled coach services will run to an altered Saturday timetable.

Metropolitan and regional bus

  • Metropolitan buses will mostly run to a Saturday timetable, with some exceptions.
  • Regional bus services will vary depending on the operator.
  • Night network services will not run.

Plan ahead by using the Journey planner or by downloading the free PTV app for real-time travel information.
